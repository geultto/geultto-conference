## 글또콘은 무엇인가요?
- 글쓰는 개발자 모임 - 글또에서 진행하는 내부 컨퍼런스입니다
- 1회차만 진행했으며, 주기나 방식은 계속 발전시킬 예정입니다
- 글또 활동 기수에 계신 분들을 대상으로만 하고 있으며, 추후에 더 확대 여부를 고민하고 있습니다
